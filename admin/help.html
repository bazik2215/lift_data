📋 Список 📊 Статистика ❓ Справка
❓ Добро пожаловать в админ-панель
Админ-панель предназначена для управления домами, лифтами, программами ремонта и краткосрочными планами. Все изменения сохраняются в файл data.json, который затем нужно загрузить на GitHub.
⚠️ ВАЖНО: После любых изменений не забудьте нажать «💾 Сохранить JSON» и загрузить полученный файл на GitHub. Без этого изменения не попадут на сайт!
📋 Раздел «Список»
🔍 Поиск
Введите адрес в поле поиска — таблица автоматически отфильтруется. Поиск работает в реальном времени.
⬆️⬇️ Сортировка
Кликните на заголовок таблицы (ID, Адрес, Район, Год) — таблица отсортируется. Повторный клик меняет направление.
✅ Выделение домов
Отметьте чекбоксы у нужных домов, затем нажмите «🗑️ Удалить выбранные» — удалятся все отмеченные дома (с подтверждением).
✏️ Редактировать
Нажмите ✏️ — откроется страница редактирования дома.
📋 Дублировать
Нажмите 📋 — создастся копия дома с новым ID. Данные останутся те же.
🗑️ Удалить
Нажмите 🗑️ — дом будет удалён после подтверждения.
➕ Добавить дом
Нажмите кнопку «➕ Добавить дом» — откроется форма для создания нового дома.
✏️ Раздел «Редактирование дома»
📋 Основная информация
Заполните адрес, район, годы постройки, этажность, серию дома, координаты.
🗺️ Координаты
Формат: 48.560115, 39.310170 (широта, долгота). Нажмите «🗺️ Показать на карте» для проверки правильности координат. Карта появляется/скрывается по нажатию.
🛗 Подъезды и лифты
Добавляйте подъезды и внутри них — лифты. У каждого лифта можно указать рег. номер, модель, год изготовления, типы, состояние и предыдущий лифт (если был заменён).
📅 Программа работ
Укажите год (например, «2024-2025») и описание ремонта.
⚡ Краткосрочный план
Укажите тип ремонта, подрядчика и период выполнения (например, «09.2025 - 12.2025»).
💾 Сохранение
Нажмите «💾 Сохранить дом» — данные сохранятся в памяти. Затем на главной странице нажмите «💾 Сохранить JSON» и загрузите файл на GitHub.
📊 Раздел «Статистика»
📈 Общие показатели
Количество домов, лифтов, программ ремонта.
📍 Распределение по районам
Сколько домов, текущих и заменённых лифтов в каждом районе.
🔧 Модели лифтов
Статистика по текущим и заменённым моделям лифтов.
🏭 Подрядчики
Анализ подрядчиков из краткосрочного плана.
💾 Сохранение и загрузка на GitHub
Пошаговая инструкция:
1. Сохраните JSON
На странице «Список» нажмите кнопку «💾 Сохранить JSON». Браузер скачает файл data.json.
2. Зайдите на GitHub
Откройте ваш репозиторий на GitHub.
3. Откройте data.json
Нажмите на файл data.json, затем на значок карандаша ✏️ (Edit).
4. Вставьте новые данные
Скопируйте содержимое скачанного data.json и вставьте вместо старого.
5. Сохраните
Нажмите «Commit changes» (зелёная кнопка).
6. Подождите
Через 1-2 минуты сайт обновится с новыми данными.
⌨️ Горячие клавиши

Esc — закрыть мини-карту
На странице входа
Enter — войти
Везде
Ctrl + / — открыть справку
⚠️ Частые ошибки и их решение
❌ Не загружается data.json
Убедитесь, что файл data.json лежит в корне репозитория, а не в папке admin.
❌ Карта не показывает дом
Проверьте координаты в формате: 48.560115, 39.310170 (широта, долгота через запятую).
❌ Не отображаются лифты
Убедитесь, что у лифта заполнено поле «Рег. номер лифта» или «Название».
❌ Изменения не видны на сайте
Возможно, вы забыли нажать «💾 Сохранить JSON» или загрузить файл на GitHub.
💾 Индикатор последнего сохранения
В правом нижнем углу всех страниц админ-панели отображается дата, время и относительное время последнего нажатия кнопки «💾 Сохранить JSON». Это помогает не забыть сохранить изменения.
